# Reference projects

This directory holds **external Godot projects and kits** checked in for local reference. Use them for patterns, APIs, scenes, and `project.godot` examples when designing blobert features.

**Godot demo projects (`godot-demo-projects/`):** **Ignore everything under `godot-demo-projects/2d/`** — blobert targets 3D scenes (2.5D), not pure 2D demos. Prefer `godot-demo-projects/3d/`, `xr/`, `viewport/`, and other top-level areas as needed.

**Do not** treat these trees as part of the blobert game project: do not add blobert gameplay code or assets here, and keep blobert’s root `project.godot` independent of paths under this folder.
