# M34: Hitbox & Frame Visualization

**Status:** Planning  
**Depends:** M11-M30

## Overview

Debug overlay system to visualize active frames and hitbox collision zones during attack development. Extract from HADS pattern and add rich visual feedback.

## Key Features

- **Frame timeline**: Current frame counter overlaid on attack duration
- **Hitbox visualization**: Collision shape display with color coding
- **Active window highlight**: Visual indication of damage application window
- **Damage vector display**: Arrow showing knockback direction
- **Performance overlay**: Frame-by-frame stats
