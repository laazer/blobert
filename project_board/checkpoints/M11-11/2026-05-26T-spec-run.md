## Key Design Decisions

1. **Root via `apply_slowness(0.0, 1.0)`** — reuses existing slowness system. No new `apply_root()` API. EnemyEffectTracker already handles `multiplier = 0.0` correctly (ADHA-DR-1).

2. **Falsy-zero fix** — changed `modifiers.get("slow", 0.0)` → `modifiers.get("slow", null)` and `if slow_val and slow_val > 0.0` → `if slow_val != null`. Applies to BOTH `AttackExecutor._apply_modifiers()` and `PlayerProjectile3D._apply_modifiers()` (ADHA-DR-2).

3. **Root + infection = tactical synergy** — rooted enemy can't dodge claw follow-up. Adhesion does NOT carry `infect_weakened: true`. Infection requires a separate claw hit (ADHA-DR-3).

4. **Wall collision despawn** — generic `else` branch in `_on_body_entered()`: any body without `take_damage()` consumes and frees the projectile. Benefits acid projectiles too (ADHA-DR-4).

5. **Projectile speed 8.0, lifetime 1.25s** — derived from ticket's 10-unit range: `10.0 / 8.0 = 1.25` (ADHA-DR-5).

6. **Color: `Color.DARK_GOLDENROD`** — amber/honey tone, distinct from CHARTREUSE (acid), ORANGE_RED (claw), SADDLE_BROWN (carapace) (ADHA-DR-6).

7. **Damage: 1.0** — CC-focused utility attack, not damage-focused (ADHA-DR-7).

## Checkpoint

### [M11-11] SPECIFICATION — Root + infection interaction mechanic
**Would have asked:** Should adhesion carry `infect_weakened: true` to auto-infect WEAKENED enemies, or should it rely on claw follow-up?
**Assumption made:** Adhesion does NOT carry `infect_weakened`. The root effect is the infection enabler — it creates a 1.0s tactical window where the enemy can't dodge a claw follow-up. This preserves distinct tactical identities: adhesion = CC, claw = infection.
**Confidence:** High

### [M11-11] SPECIFICATION — Adhesion projectile color
**Would have asked:** What specific color should the adhesion projectile use?
**Assumption made:** `Color.DARK_GOLDENROD` (amber/honey tone) — evokes stickiness and is distinct from all other attack colors (ORANGE_RED, CHARTREUSE, SADDLE_BROWN).
**Confidence:** Medium

### [M11-11] SPECIFICATION — Direct-hit damage value
**Would have asked:** What should adhesion's direct-hit damage be?
**Assumption made:** 1.0 damage — adhesion is a CC-focused utility attack. Its value is the root window, not raw damage. Matches acid's direct-hit damage for projectile parity.
**Confidence:** High
