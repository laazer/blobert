# M20: Active Ability UI System

**Status:** Planning  
**Depends:** M11-M12

## Overview

HUD system showing current active mutations, cooldown timers, charge meters, and combo feedback. Integrated with PlayerController and mutation slot system.

## Key UI Elements

- **Mutation indicator**: Shows current mutation(s) and which slot
- **Cooldown timer**: Circular or linear display, one per mutation
- **Charge meter**: Fills as button is held (M14 integration)
- **Combo counter**: Feedback for hit combos and cancel windows
- **Attack name/description**: Brief info when switching mutations
