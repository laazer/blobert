# M19: Ability Progression & Leveling

**Status:** Planning  
**Depends:** M11-M12, M18

## Overview

Mutations can be leveled up by absorbing the same mutation multiple times. Each level increases damage, cooldown efficiency, and unlocks new modifiers or variants.

## Key Mechanics

- **Level scaling**: Damage, cooldown, range scale per level
- **Modifier unlocks**: Higher levels unlock new modifiers (e.g., Claw level 3 adds armor pen)
- **Visual progression**: Color tint, particle intensity indicate mutation level
- **Inventory tracking**: Per-mutation level state
