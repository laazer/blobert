# Light skill check

**Epic:** Milestone 4 – Prototype Level  
**Status:** Backlog

---

## Description

Add a light skill check (e.g. platforming, timing, or simple combat) that uses movement and optionally one mutation so the level has a moment of challenge without being a full boss.

## Acceptance criteria

- [ ] Skill check is passable with core movement (and optionally one mutation)
- [ ] Failure has clear consequence (e.g. damage, retry) and no softlock
- [ ] Difficulty is “light”: completable in a few attempts by target audience
- [ ] Placed appropriately in level flow (e.g. before or after fusion)
