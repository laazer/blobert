# Start → Finish flow

**Epic:** Milestone 4 – Prototype Level  
**Status:** Backlog

---

## Description

Ensure full start-to-finish flow: player spawns at start, plays through mutation tease, fusion opportunity, skill check, and mini-boss, then reaches level end. No debug tools required.

## Acceptance criteria

- [ ] Level is completable from start to finish without debug or cheats
- [ ] Flow is clear (signposting or layout so player knows direction)
- [ ] Target 6–8 minutes playtime for first full run
- [ ] All milestone 4 scope items are integrated and playable
