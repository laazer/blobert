## Description

The asset_generation/python pipeline already generates animated enemies via
`python main.py animated <type> <variants>`. Three enemy types exist:
adhesion_bug, tar_slug, ember_imp.

M5 requires 4 families: adhesion (done), acid, claw, carapace.

This ticket adds the 3 missing animated enemy classes to the existing pipeline
and generates the GLB files for each (3 variants per family = 9 new GLBs).

## Files to Modify / Create

- `src/enemies/animated_enemies.py` — Add AnimatedAcidSpitter, AnimatedClawCrawler, AnimatedCarapaceHusk classes; register all 3 in AnimatedEnemyBuilder.ENEMY_CLASSES
- `src/utils/constants.py` — Move ACID_SPITTER and CLAW_CRAWLER from get_static() to get_animated(); add CARAPACE_HUSK = "carapace_husk"
- `src/utils/materials.py` — Add carapace_husk material theme to MaterialThemes.ENEMY_THEMES
- `tests/enemies/test_animated_enemy_classes.py` — New pure-Python test file (no Blender required)

## Existing Infrastructure Notes

- Material themes for acid_spitter and claw_crawler already exist in src/utils/materials.py
- carapace_husk needs a new theme entry (suggested: STONE_GRAY, BONE_WHITE, CHROME_SILVER)
- Body type guide: AcidSpitter -> BLOB (ranged, pulsing), ClawCrawler -> QUADRUPED (ground, pounce), CarapaceHusk -> HUMANOID (heavy, charge)
- After classes are added, GLBs are generated by running: python main.py animated acid_spitter 3, etc.
- Output lands in animated_exports/
- Attack profiles in src/combat/enemy_attack_profiles.py are NOT required for this ticket (out of scope)

## Body Geometry Notes (from Planner planning)

- AnimatedAcidSpitter: BLOB type, pulsing sac body (sphere with different scale ratios from TarSlug), reuses create_blob_armature
- AnimatedClawCrawler: QUADRUPED type, low flat body with 2 front "claw" cylinders (wider/shorter) replacing front legs, 4 regular legs, reuses create_quadruped_armature
- AnimatedCarapaceHusk: HUMANOID type, heavier/wider cylindrical body than EmberImp, shorter legs, arm protrusions instead of hands, reuses create_humanoid_armature
